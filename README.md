# 💬 项目名称  
**women俩｜母女沟通与共听播客的小程序平台**

---

## 🧠 项目简介  
「women俩」是一个专为母女沟通设计的小程序平台，帮助妈妈与女儿在一起听播客的过程中自然展开对话，修复代际沟通的隔阂。  
它不仅让两代女性可以“边听边聊”，还会自动整理她们的互动内容，生成属于母女二人的播客回顾节目。

---

## 👥 目标用户  

### 用户画像  
- **主要用户：** 母亲与女儿（跨代女性群体）  
- **核心场景：** 通过播客共听与互动，促进母女之间的理解与表达  

### 用户痛点  
- 母女间沟通匮乏，常被生活与代际观念差异阻断  
- 缺乏情感对话的场域，不知道如何开启深入话题  
- 共听播客时缺乏沉浸互动体验，听完后难以保留共鸣内容  

### 使用场景  
- 晚上休息或通勤途中，母女远程共听播客、语音交流  
- 家庭时光中，母女共同回顾过去的对话与心情  
- 听完播客后整理与分享自制的「母女回顾播客」  

---

## ⚙️ 功能规格  

### 🌟 一级核心功能：共听与实时互动  
- **一起听播客并实时互动**：母女双方可同时播放同一播客内容  
- **实时语音沟通**：在听的过程中随时长按对话按钮开麦对谈  
- **表情与文字弹幕**：同步发出表情或评论，增强陪伴感与互动感  
- **Casual模式与正式对谈模式切换**：  
  - *Casual 模式*：类似网易云“一起听”的随意语音交流  
  - *正式对谈模式*：播放页下拉后点击“开启正式对谈会议室”，跳转至类似腾讯会议的语音对谈界面  

---

### 🎧 二级功能：互动回顾与播客生成  
- **互动记录回顾**：支持回听对话音频与查看文字互动记录  
- **自动生成回顾播客**：系统可将一次共听记录加工为完整播客节目  
  - 内容包含：  
    - 播客原名称  
    - Shownotes（对话要点）  
    - 加入背景音乐与编辑的音频文件  
    - 文字评论自动生成弹幕回放效果  
- **情感记录功能**：记录母女间的互动次数、话题关键词与情感趋势  

---

### 🪄 三级功能：播客推荐与共听邀请  
- **播客推荐页**：精选女性向播客主题，用户可直接播放  
- **直接播放与共听邀请**：  
  - 点击播客卡片（举例首推《鲁豫对话小帕｜要走多少路，才能成为一个女人》）可立即播放  
  - 支持一键邀请对方共同收听  
- **听后感与讨论区**：用户可写下听后感或留言评论  
- **推荐案例**：  
  1. 鲁豫对话小帕｜要走多少路，才能成为一个女人  
  2. 如何与更年期老妈相处：理解我妈，调查外婆，研究激素  
  3. 与海淀妈妈聊天：理想中的主体性，现实里的鸡娃战  
  4. 电影解析播客推荐：《坠落的审判》《瞬息全宇宙》《好东西》《女人世界》《我的阿勒泰》等

## 🧩 开发规划  

### 技术架构  
- **前端：** html网页 （方便放到h5托管平台进行demo测试）
- **后端：** Node.js + Express + WebSocket（实时语音与弹幕通信）  
- **数据库：** MySQL（用户与互动数据）+ OSS（音频文件存储）  
- **音频处理：** FFmpeg 自动生成播客节目  
- **AI 模块：** Whisper + ChatGPT（对话摘要与 shownotes 自动生成）  

---
